Code to unsubscribe from all channels on your youtube account.

YouTube does not have a one-click button to unsubscribe from all channels at once. 
Must run a script to remove it automatically one-by-one. Follow below instructions.

Working as of 12 March 2026.

Steps:
1. Open Firefox
2. Open Youtube and login to account
3. Go to www.youtube.com/feed/channels
4. Press F12 on Keyboard. 
5. Click on Console. 
6. Copy paste code in remove_subscriptions.js on the console next to >>
7. Press Enter.
